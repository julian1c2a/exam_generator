# Requisitos del Proyecto Exam Generator
# 
# Este proyecto NO requiere dependencias externas.
# Solo necesita Python 3.9+ y sus librerías estándar.
#
# Las librerías usadas son todas incluidas en Python:
# - argparse (CLI)
# - json (serialización)
# - sqlite3 (base de datos)
# - pathlib (gestión de rutas)
# - shutil (operaciones de archivos)
# - datetime (timestamps)
# - logging (logs)
# - collections (utilidades)
# - enum (enums)
# - dataclasses (dataclasses)
# - typing (type hints)
#
# Para instalar el proyecto:
# 1. Clonar repositorio:
#    git clone https://github.com/julian1c2a/exam_generator.git
#
# 2. Cambiar a rama GeneratorFEExercises:
#    git checkout GeneratorFEExercises
#
# 3. Usar directamente (sin instalación de paquetes):
#    python -m cli --help
